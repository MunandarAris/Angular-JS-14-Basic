Pipes sendiri memiliki beberapa function, tetapi disini saya hanya mencantumkan yang menurut saya paling sering di gunakan :

  • DatePipe => untuk merubah format tanggal dari database dan ingin di sesuaikan di frontend nya
  • KeyValue => jika kita ingin melakukan loop data dan ternyata antara key dan nilainya sama, kita bisa meanfaatkan ini
  • TitleCasePipe => jika kita ingin menampilkan text ke frontend menjadi " Makanan Adalah Makanan "
  • SlicePipe => ini fungsinya untuk melakukan slice data dari sebuah array
  • PercentPiep => mengubah nilai desimal ke dalam persen
  • CurrencyPipe => memberikan matauang
